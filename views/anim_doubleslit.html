Double-Slit Interference

About this phenomenon

Thomas Young's double-slit experiment demonstrates the wave-particle duality. Unobserved, particles create an interference pattern (Wave Nature). However, when an Observer measures which slit the particle passes through, the wavefunction collapses and the interference pattern disappears (Particle Nature).
